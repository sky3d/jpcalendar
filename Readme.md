### FAQ

1. How to set ANDROID_SDK_ROOT in mac?

```
vim ~/.bash_profile
Then add the following environment variables:

export ANDROID_HOME=~/Library/Android/sdk
export ANDROID_SDK_ROOT=~/Library/Android/sdk
export ANDROID_AVD_HOME=~/.android/avd
```
